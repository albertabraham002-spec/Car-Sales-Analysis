A Power BI dashboard project analyzing car sales and pricing data to identify trends and key factors affecting vehicle prices across brands, fuel types, transmission, and car condition.

Dataset: Kaggle (2,500 records, 28 models)
Tools: Power BI, Power Query, DAX

Key Insights

Premium models contribute the highest total value

Petrol and Diesel dominate the market

Automatic cars generate higher overall value

Electric and Hybrid cars show higher average prices
